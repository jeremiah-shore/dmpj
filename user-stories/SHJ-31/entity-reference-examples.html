Certain special characters have reserved meanings or uses in HTML and cannot always be used on their own in an HTML file. They can, however, be displayed using what's called "character references". A character reference is a sequence of characters that starts with an ampersand or &, is filled with a code of alpha-numeric characters, and ends with a semicolon.

A couple example character references (aka entity references):

  • A '<' or "less than character" can be displayed with a character reference of &lt;
  • A '>' or "greater than character" can be displayed with a character reference of &gt;
